A feature-rich Redbox inventory software for efficiently managing Redbox kiosk inventory, titles, and more.

redbox-apis

A feature-rich API designed to restore the functionality of your Redbox kiosk, as well as Redbox Perks. An attempt to reverse engineer the Redbox backend APIs.

Author(s): Puyodead1, Brian Walczak, Legoman99573
GitHub ➔

HW-Emu-Python

Python-based emulator for kiosk hardware, simulating devices the HAL communicates with. Useful for development and testing without physical hardware.

Author(s): Puyodead1

findaredbox
